## 🧊 О проекте

**CodeIgloo** — это холодное хранилище для твоего кода, созданное в рамках организации **TPickME**. Здесь нет рекламы, нет трекеров, нет лишних вопросов. Просто вставь код в морозное поле, нажми «Сохранить» — и получи короткую ссылку, которую можно отправить куда угодно.

Подходит для:
- 🐍 быстрого шаринга кода в чатах
- 📦 временного хранения сниппетов
- 🤝 передачи примеров друзьям и коллегам
- 🧊 поддержания зимнего вайба даже в коде

<br>

## 📦 Как это работает

1. Вставляешь код в ледяное поле
2. Нажимаешь «Сохранить в Igloo»
3. Получаешь ссылку, которая ведёт прямо на этот код
4. Отправляешь ссылку другу — всё

**Технически:** код не хранится на сервере, а вшивается прямо в URL (через base64). Никаких баз данных, никакой подписки. Просто и прозрачно.
